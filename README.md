# Be the hero

## Sobre o Projeto:
### Be the hero se trata do projeto Oministack, onde é trabalhado Node.js, React e React Native, juntando eles em um serviço onde se é possivel cadastrar ONGs e casos de Ongs que precisam de ajuda, atravez da apliação mobile é possivel entrar em contato com a ong atravez do email ou whatsapp
